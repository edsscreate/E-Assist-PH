# E-Assist PH Website

A responsive, one-page static website for **E-Assist PH — One-Stop Digital Assistance**.

## Files
- `index.html` — website content
- `styles.css` — design and responsive layout
- `script.js` — menu, FAQ, service selection, and inquiry-message generator

## Open the website
1. Extract the ZIP file.
2. Open the folder.
3. Double-click `index.html`.

## Important edits before publishing
Open `index.html` using Notepad, VS Code, or any code editor.

### Change contact links
Search for:
```html
<a href="#" data-placeholder-link>💬 Facebook Messenger <span>↗</span></a>
```

Replace `href="#"` with your real links. Examples:
```html
<a href="https://m.me/YOUR_PAGE_USERNAME" target="_blank">💬 Facebook Messenger <span>↗</span></a>
<a href="mailto:your@email.com">📧 Email us <span>↗</span></a>
<a href="tel:+639123456789">📱 Call / Text us <span>↗</span></a>
```

### Change service details
Edit the text inside the Services and FAQ sections of `index.html`.
